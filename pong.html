Ping Pong Project Computer science

The objective of the Ping Pong project is to code and engineer a contraption that will launch the ping pong ball go over the net. For my project, I used Arduino for coding and engineered the catapult with the inspiration of a medieval catapult. With the help of my partner,Bryan Jiron, we were able to complete this project successfully. I learned how to incorporate coding into an engineering aspect and also how to modify a code that was created by others. If I had more time for my project, I would've improved the distance that the catapult launched the ball.

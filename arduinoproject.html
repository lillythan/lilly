House TinkerCad

For this house project, I initally started the tutorial on how to use TinkerCad and build basic shapes. The main aspect of this project was focusing on engineering and creativity. I was inspired by an original house and used my imagination from there on. The challenging part of this project was small details of the house. However, I some-what enjoyed designing a house out of creativity.
